Global Header Component

Material Design 3 dark theme header with Blue (#2196f3) primary and Orange (#ff9800) accent colors. This component provides consistent navigation across the application.

Component Features

  • check_circle Logo & Branding - Blue/Orange gradient icon, collapsible text
  • check_circle Menu Items - File, Edit, View, Go, Run, Help (hidden on mobile)
  • check_circle Search Bar - Centered search with keyboard shortcut indicator
  • check_circle Action Buttons - Notifications, Account icons
  • check_circle Window Controls - Minimize, Maximize, Close (desktop only)
  • check_circle Mobile Menu - Hamburger button replaces menu on small screens

Design Tokens Used

Primary: #2196f3 (Blue)

Accent: #ff9800 (Orange)

Background: #121212

Surface: #1e1e1e (Cards/Toolbar)

Elevated: #2d2d2d (Inputs/Elevated)

Header Height: 64px

Responsive Behavior

Mobile (XS-SM): Hamburger menu, compact logo, search centered

Tablet (MD): Full menu visible, logo text shown

Desktop (LG+): All elements visible including window controls
