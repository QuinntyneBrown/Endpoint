Rail Component (Activity Bar)

VS Code-inspired vertical navigation bar with Angular Material icons. Fixed on the left side, providing quick access to main application areas.

Component Features

  • check_circle Fixed Position - Always visible on left side (desktop/tablet)
  • check_circle Icon Navigation - Material icons for each section
  • check_circle Active Indicator - Primary color bar on active item
  • check_circle Badge Support - Notification badges for counts
  • check_circle Hover Tooltips - Labels appear on hover
  • check_circle Settings at Bottom - Gear icon anchored to footer

Rail Icons

folder Explorer

Responsive Behavior

Mobile (XS-SM): Rail is hidden. Access via hamburger menu overlay.

Tablet (MD): 48px collapsed rail visible with icons only.

Desktop (LG+): 48px rail with optional expanded sidebar (240px).

Angular Material Components Used

mat-icon-button, mat-icon, mat-badge, mat-tooltip, mat-sidenav (for expanded state)
