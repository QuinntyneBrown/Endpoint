Rail Component (Activity Bar)

Material Design 3 vertical navigation bar with Blue (#2196f3) primary and Orange (#ff9800) accent. Fixed on the left side, providing quick access to main application areas.

Component Features

  • check_circle Fixed Position - Always visible on left side (desktop/tablet)
  • check_circle Icon Navigation - Material Icons Outlined for each section
  • check_circle Active Indicator - Blue (#2196f3) bar on active item
  • check_circle Badge Support - Orange (#ff9800) badges for counts
  • check_circle Hover Tooltips - Labels appear on hover
  • check_circle Settings at Bottom - Gear icon anchored to footer

Design Tokens Used

Primary: #2196f3 (Blue - active indicator)

Accent: #ff9800 (Orange - badges)

Background: #121212

Surface: #1e1e1e (Rail background)

Rail Width: 48px

Status Bar: #1976d2 (primary-700)

Responsive Behavior

Mobile (XS-SM): Rail is hidden. Access via hamburger menu overlay.

Tablet (MD): 48px collapsed rail visible with icons only.

Desktop (LG+): 48px rail with optional expanded sidebar (240px).
