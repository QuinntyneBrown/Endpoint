# Endpoint.React

The React code generation module for the Endpoint framework. This library provides generators for creating React components, hooks, and related frontend artifacts.

## Overview

Endpoint.React enables the generation of React frontend code that integrates with backend services generated by other Endpoint modules. It supports modern React patterns and best practices.

## Key Features

- **Component Generation**: Functional React components with hooks
- **Hook Generation**: Custom React hooks for state and effects
- **TypeScript Support**: Full TypeScript type generation
- **Modern Patterns**: Support for React 18+ patterns

## Usage

Endpoint.React is typically used through the Endpoint.Cli:

```bash
# Create a React application
endpoint react create --name MyApp

# Generate a React component
endpoint react component --name CustomerList

# Generate a custom hook
endpoint react hook --name useCustomers
```

## Generated Artifacts

- Functional components with TypeScript
- Custom hooks (useState, useEffect patterns)
- Context providers
- API integration hooks

## Integration with Backend

Generated React components can integrate with APIs generated by Endpoint.DotNet:

```typescript
// Generated hook
const useCustomers = () => {
  const [customers, setCustomers] = useState<Customer[]>([]);

  useEffect(() => {
    fetchCustomers().then(setCustomers);
  }, []);

  return { customers };
};
```
